Лабораторная работа №4 

Исходный сайт: https://shop.mango.com/ru

План тест-кейсов (он не совсем необычный, но показывает поведение пользователя, который редко заходит на подобные сайты):

1) Поиск товара по ключевому слову;
2) Использование фильтров для более узкого выбора;
3) Добавление товара в раздел "Избранное";
4) Добавление товара в корзину;
5) Удаление товара из корзины;
6) Переход в социальную сеть данного магазина из футтера сайта.

Я искал в интернете возможности Селениума и обнаружил интересную библиотеку "selenium.webdriver.common.by", которая позволяет ображаться к элементам веб-страницы через "XPath" (XML-ный язык запросов, позволяющий работать с элементами HTML-кода). 
Процесс простой - мы создаем запрос, описывающий необходимые данные, и затем его получаем через "XPath".

Исходный код был приложен в репозитории, а работает автоматически.
